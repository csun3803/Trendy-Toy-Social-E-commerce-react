# 潮玩社交电商系统 - 活动图说明文档

## 文件说明

`TrendyToyActivityDiagrams.puml` - 使用PlantUML语法编写的系统活动图文件，包含9个核心业务流程的活动图。

## 活动图列表

### 1. 潮玩社交电商系统 - 主要业务流程总图
- **概览**：展示系统三大核心模块的整体业务流程
- **包含内容**：
  - 用户端流程
  - 商家端流程
  - 平台管理流程

### 2. 用户注册登录活动图
- **流程**：
  1. 启动App，检查登录状态
  2. 输入账号密码，勾选协议
  3. 发送登录请求
  4. 保存token，跳转首页
- **支持方式**：
  - 账号密码登录
  - 微信登录
  - QQ登录
  - 微博登录

### 3. 商品浏览购买活动图
- **流程**：
  1. 浏览商品列表/查看商品详情
  2. 立即购买或加入购物车
  3. 确认订单信息
  4. 选择支付方式并完成支付

### 4. 订单管理活动图
- **订单状态**：
  - PENDING（待付款）
  - PENDING_SHIPMENT（待发货）
  - SHIPPED（待收货）
  - COMPLETED（已完成）
  - CANCELLED（已取消）
- **操作**：付款、取消订单、确认收货、评价商品

### 5. 展示柜管理活动图
- **功能**：
  - 查看展示柜列表
  - 添加潮玩到展示柜
  - 创建新展示柜
  - 设置展示柜权限
  - 分享展示柜

### 6. 社交活动发布流程活动图
- **流程**：
  1. 填写活动信息
  2. 上传活动图片
  3. 设置活动权限
  4. 提交审核
  5. 审核通过后发布

### 7. 商家入驻审核活动图
- **流程**：
  1. 商家填写入驻申请
  2. 上传资质文件（营业执照、法人身份证等）
  3. 平台管理员审核
  4. 审核通过后商家可正常营业

### 8. 商家商品管理活动图
- **功能**：
  - 查看商品列表
  - 添加新商品
  - 编辑商品信息
  - 管理商品库存
  - 设置商品价格
  - 下架商品

### 9. 商家端订单处理流程活动图
- **流程**：
  1. 查看待发货订单
  2. 确认订单和收货地址
  3. 填写物流信息并发货
  4. 订单流转至完成状态

### 10. 用户消息与互动流程活动图
- **功能**：
  - 查看消息列表
  - 发送接收消息
  - 关注其他用户
  - 点赞、评论、收藏社交活动

## 如何使用

### 1. 使用在线PlantUML编辑器
- 访问 [PlantText](https://www.planttext.com/) 或 [PlantUML在线编辑器](http://www.plantuml.com/plantuml/)
- 复制 `TrendyToyActivityDiagrams.puml` 文件的内容并粘贴
- 点击生成图表

### 2. 使用VSCode插件
- 安装 [PlantUML插件](https://marketplace.visualstudio.com/items?itemName=jebbs.plantuml)
- 在VSCode中打开 `TrendyToyActivityDiagrams.puml`
- 按 `Alt+D` 预览图表

### 3. 使用命令行工具
- 安装PlantUML
- 运行: `plantuml TrendyToyActivityDiagrams.puml`
- 生成的PNG/SVG图片将保存在同一目录

## 业务角色说明

### 1. 普通用户
- 浏览和购买潮玩
- 管理购物车和订单
- 创建展示柜展示收藏
- 发布和参与社交活动
- 发送私人消息

### 2. 商家
- 申请入驻平台
- 管理店铺信息
- 发布和管理商品
- 处理订单和发货

### 3. 平台管理员
- 审核商家入驻申请
- 审核社交活动内容
- 管理用户账户

## 核心业务闭环

1. **商品流**：商品发布 → 用户浏览 → 下单 → 支付 → 发货 → 收货 → 评价
2. **社交流**：用户注册 → 发布活动 → 审核通过 → 其他用户互动 → 分享展示柜
3. **商家流**：申请入驻 → 审核通过 → 发布商品 → 订单处理 → 发货 → 结算
